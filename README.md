# Social Media Analytics Report

## Project Overview

This project analyzes social media performance data for a fashion brand using Microsoft Excel and Power BI. The goal of the project is to understand platform-wise performance, post type engagement, reach, impressions, followers gained, and link clicks.

## Tools Used

* Microsoft Excel
* Power BI
* GitHub

## Dataset

The dataset contains social media analytics data with the following columns:

* Date
* Platform
* Post Type
* Likes
* Comments
* Shares
* Reach
* Impressions
* Followers Gained
* Link Clicks
* Engagement Rate

## Key Insights

* Instagram generated the highest reach and impressions, making it the best platform for brand visibility.
* Reels performed better than other post types in terms of likes and engagement.
* LinkedIn had lower reach compared to Instagram and Facebook, but showed better link click performance.
* Facebook showed balanced performance but did not outperform Instagram in reach and engagement.
* Short-form video content such as Reels can help improve reach, followers gained, and engagement rate.

## Conclusion

The analysis shows that Instagram is the strongest platform for brand awareness, while LinkedIn is useful for driving link clicks. The brand should focus more on Instagram Reels to improve engagement and visibility.
